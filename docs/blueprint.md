# **App Name**: StepSafe AI

## Core Features:

- Location Detection: Automatically detect the user's location or allow manual input for weather and air quality data retrieval.
- Climate Data Fetch: Retrieve real-time weather data (temperature, humidity, rain probability, UV Index) and Air Quality Index (AQI) from reliable APIs.
- Rule-Based Risk Engine: Convert raw climate data into risk levels (Low, Medium, High, Extreme) for heat, UV, AQI, humidity, and rain exposure using deterministic calculations.
- AI Safety Advisory: Generate a concise, human-readable safety advisory based on calculated risk levels using a large language model as a tool to produce the text.
- Personalized Checklist: Generate a 'Before You Step Out' checklist with actionable recommendations for water intake, clothing, sunscreen, and mask usage.
- Safe Time Windows: Identify safe and unsafe outdoor time windows based on risk levels, displayed on a clear, visual timeline.
- User Profile (Optional): Allow users to input optional profile information (age group, activity level) to personalize risk assessments and recommendations.

## Style Guidelines:

- Primary color: Soft blue (#A7D1AB), evoking trust and health. It's analogous to teal, as the user requested, without being literally teal.
- Background color: Light gray (#F0F0F0), provides a clean and calm backdrop.
- Accent color: Pale green (#C5E898), to be used to direct the user's eye, analogous to soft blue.
- Body and headline font: 'Inter', sans-serif, for high readability and a modern feel.
- Card-based layout, mobile-first design, fully responsive for desktop and mobile.
- Use clear, simple icons to represent different risk factors and recommendations.
- Subtle animations to guide user and highlight key information
